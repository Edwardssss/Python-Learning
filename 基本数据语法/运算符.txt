# 算术运算符
# +，-，*，/ （加，减，乘，除）
# % 取模：左边数除以右边数的余数
# ** 幂 a**b a的b次幂
# //整除：返回商的整数部分

# 赋值运算符
# = 右边的值赋给左边
# +=，-=，*=，/=，%=，**=，//= 先运算，再赋值 a+=b 等价于 a=a+b

# 比较运算符
# == 比较是否相等
# != 比较是否不相等
# >,<,>=,<= 大于小于

# 位运算符
# &，|，^ ,~ 按位与，或，异或（对应值相同则为0，否则为1），取反
# <<,>> 按位左移，右移

# 逻辑运算符
# and,or not 与 或 非

# is和in
# is用于判断两个标识符是否引用于同一个对象、
# in判断是否包含在指定的序列当中
# 注意：is的判断引用对象是否为同一个，==判断值是否相等

# 可以用 " # " 来作为注释的开头（仅限一行）
