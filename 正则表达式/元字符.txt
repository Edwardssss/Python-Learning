# \  将下一个字符标记为特殊字符或者字面值
# ^  匹配输入的开始部分
# $  匹配输入的结束部分
# *  匹配零次或多次前面的字符
# +  匹配至少一次前面的字符
# ?  匹配零次或一次前面的字符
# .  匹配任何单个的字符（除换行符之外）
# x|y  匹配x或y
# {n}  n为一个非负整数，表示匹配n次前面的字符（精确，不多不少）
# {n,}  表示至少匹配n次前面的字符，只多不少
# {n,m}  表示匹配至少n次至多m次前面的字符
# [xyz]  表示匹配任意一个包含其中的字符
# [^xyz]  表示否定，除包含的字符以外的所有字符
# [a-z]  表示匹配字符的范围，匹配指定范围内的所有字符
# [^a-z]  表示否定，匹配除字符范围以内的所有字符
# \A  仅匹配字符串的开头
# \b  匹配单词的边界，即单词和空格之间的位置
# \B  匹配非单词的边界，也就是可以插入单词
# \d  匹配数字字符
# \D  匹配非数字字符
# \f  匹配换页字符
# \n  匹配换行符
# \r  匹配回车字符
# \s  匹配任何空白字符，包括空格，制表符，换页字符等等
# \S  匹配任何非空白字符
# \t  匹配跳进字符
# \v  匹配垂直跳进字符
# \w  匹配任何数字，单词字符，包括下划线
# \W  匹配任何非数字，单词字符
# \z  仅匹配字符串的结尾
# \Z  仅匹配字符串的结尾，或者结尾的换行符之前
